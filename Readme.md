本文档维护了中山大学计算机学院SMCLab的日常事务（基于飞书API）的爬取，包括**学生信息、组会维护、周报抓取、考勤记录**等，欢迎SMC实验室的同学一起来维护。
本仓库为非营利、永久免费的开源项目，希望对任何基于飞书进行管理的实验室甚至企业提供参考。如果帮助到了你，希望给一个Star🌟
